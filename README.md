# Sunitha Dasari — Portfolio

A static portfolio site. No build step needed.

## Deploy to Vercel

**Option A — Drag & drop (easiest):**
1. Go to https://vercel.com/new
2. Drag this whole folder (after unzipping) into the upload area
3. Click Deploy — done!

**Option B — Vercel CLI:**
```
npm i -g vercel
cd portfolio
vercel --prod
```

## Files
- index.html   — the entire site (HTML + CSS + JS in one file)
- assets/sunitha.jpg — profile photo
- vercel.json  — Vercel config (clean URLs)
